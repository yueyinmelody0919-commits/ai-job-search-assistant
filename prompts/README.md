# Prompts & Conversation Exports

This directory contains Claude Code conversation exports documenting the prompting history used to build this project.

## How to Export Claude Code Conversations

In Claude Code CLI:
- Use `/export` to export the current conversation
- Conversations are saved as JSON/Markdown files

## Contents

- Conversation exports will be added as the project progresses
- Key prompts and decisions are also documented in `BUILD_LOG.md`
